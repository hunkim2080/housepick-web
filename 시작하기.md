# 웹사이트 시작 가이드

## 빠른 시작

1. **의존성 설치**
```bash
cd website
npm install
```

2. **서버 실행**
```bash
npm start
```

3. **브라우저에서 접속**
- 홈페이지: http://localhost:3001
- 관리자 페이지: http://localhost:3001/admin/login
  - 사용자명: `admin`
  - 비밀번호: `admin123`

**포트 변경 방법:**
- 환경 변수로 설정: `PORT=8080 npm start`
- 또는 `server.js` 파일에서 `const PORT = 3001;` 부분을 원하는 포트로 변경

## 주요 기능

### ✅ 구현 완료

1. **홈페이지**
   - 서비스 소개
   - 신뢰 요소 표시
   - 고객 후기
   - 패키지 할인 배너

2. **서비스 페이지**
   - 줄눈시공
   - 입주청소
   - 탄성코트
   - 화장실 리모델링

3. **지역별 페이지**
   - 서울, 경기, 인천, 부산
   - 지역별 서비스 페이지

4. **온라인 예약 시스템**
   - 단계별 예약 프로세스
   - 견적 자동 계산
   - 패키지 할인 적용
   - 계약금 계산

5. **관리자 페이지**
   - 대시보드 (통계)
   - 예약 관리
   - 후기 관리 (승인/거부)

## 프로젝트 구조

```
website/
├── server.js              # Express 서버
├── package.json           # 의존성 관리
├── views/                 # EJS 템플릿
│   ├── layout.ejs        # 공통 레이아웃
│   ├── index.ejs         # 홈페이지
│   ├── service.ejs       # 서비스 페이지
│   ├── reservation.ejs   # 예약 페이지
│   ├── region.ejs        # 지역 페이지
│   └── admin/            # 관리자 페이지
├── public/               # 정적 파일
│   ├── css/
│   │   ├── style.css    # 메인 스타일
│   │   └── admin.css    # 관리자 스타일
│   └── js/
│       ├── main.js      # 메인 스크립트
│       └── reservation.js # 예약 스크립트
└── data/                 # 데이터베이스 (JSON)
    ├── reservations.json
    ├── reviews.json
    └── settings.json
```

## 문제 해결

### 포트 변경하기

**방법 1: 환경 변수 사용**
```bash
# Windows (PowerShell)
$env:PORT=8080; npm start

# Windows (CMD)
set PORT=8080 && npm start

# Mac/Linux
PORT=8080 npm start
```

**방법 2: server.js 파일 수정**
```javascript
// server.js 파일에서
const PORT = process.env.PORT || 8080; // 원하는 포트 번호로 변경
```

### 포트가 이미 사용 중인 경우
```bash
# Windows
netstat -ano | findstr :3001
taskkill /PID [PID번호] /F

# Mac/Linux
lsof -ti:3001 | xargs kill
```

### 모듈을 찾을 수 없는 경우
```bash
npm install
```
